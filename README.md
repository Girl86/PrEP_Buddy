# PrEP Buddy 🤖💊

PrEP Buddy is an AI-powered chatbot designed to improve awareness and access to HIV prevention information, including PrEP (oral and injectable), especially for young people.

It provides simple, safe, and friendly explanations about HIV prevention options and helps users understand eligibility, usage, and access pathways.

---

## 🚀 Features

- 💬 AI-powered chatbot using Groq LLM
- 🧠 PrEP-focused health education assistant
- 🌍 Youth-friendly HIV prevention awareness tool
- ⚡ Fast Flask backend API
- 🌐 Simple HTML/CSS/JavaScript frontend
- 🔗 Frontend–backend integration via REST API

---

## 🛠️ Tech Stack

- Backend: Python, Flask, Flask-CORS
- AI: Groq API (Llama models)
- Frontend: HTML, CSS, JavaScript
- Environment: Python virtual environment (venv)
